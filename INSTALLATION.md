# Installation et Lancement

## Étapes pour démarrer le projet

Depuis le terminal Linux (WSL), exécutez les commandes suivantes :

```bash
# Naviguer vers le projet
cd "/mnt/c/Users/dynas/Documents/Les Ombres du Son"

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

Le site sera accessible sur `http://localhost:4321`

## Commandes disponibles

- `npm run dev` - Démarre le serveur de développement
- `npm run build` - Construit le site pour la production
- `npm run preview` - Prévisualise la version de production
- `npm run astro` - Exécute les commandes Astro CLI
